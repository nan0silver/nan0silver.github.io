[TIL] Springм—җм„ң interfaceмҷҖ abstract class

28 Mar 2025 in TIL on Til, Spring Last modified at: 2025-04-08

2025-03-28 TIL

рҹ”— мӣҗліё мқҙмҠҲ: #33
рҹ“… мһ‘м„ұмқј: 2025-03-28
рҹ”„ мөңмў… мҲҳм •: 2025л…„ 04мӣ” 08мқј

рҹҚҖ мғҲлЎӯкІҢ л°°мҡҙ кІғ

  • Springмқ„ кіөл¶Җн•ҳлӢӨк°Җ interfaceмҷҖ abstract class к°ңл…җмқҙ н—·к°Ҳл Өм„ң л‘ҳ мӮ¬мқҙм—җ м–ҙл–Ө кҙҖкі„к°Җ мһҲлҠ”м§Җ м•Ңм•„ліҙм•ҳлӢӨ.

    мқён„°нҺҳмқҙмҠӨлҘј м¶”мғҒ нҒҙлһҳмҠӨк°Җ мғҒмҶҚл°ӣлҠ” кІҪмҡ°лҠ” кі кёүк°ңл…җмқҙлқјкі  н•ңлӢӨ..

вң… мқён„°нҺҳмқҙмҠӨ(interface)лһҖ?

  • мқјмў…мқҳ кё°м•Ҫ(м•ҪмҶҚ). м–ҙл–Ө кё°лҠҘмқ„ л°ҳл“ңмӢң кө¬нҳ„н•ҙм•ј н•ңлӢӨлҠ” кұё лӘ…мӢңн•ҙмЈјлҠ” кІғ.
  • мҳҲ: interface Animal { void sound(); }
    вҶ’ Animal мқён„°нҺҳмқҙмҠӨлҘј кө¬нҳ„н•ҳлҠ” нҒҙлһҳмҠӨлҠ” sound()лҘј л°ҳл“ңмӢң кө¬нҳ„н•ҙм•ј н•Ё.

вң… м¶”мғҒ нҒҙлһҳмҠӨ(abstract class)лһҖ?

  • мқјл¶ҖлҠ” кө¬нҳ„лҗҳм–ҙ мһҲкі , мқјл¶ҖлҠ” кө¬нҳ„мқҙ м•Ҳ лҗң мӨ‘к°„ лӢЁкі„мқҳ нҒҙлһҳмҠӨ
  • кіөнҶө кё°лҠҘмқҖ кө¬нҳ„н•ҳкі , н•„мҲҳ кө¬нҳ„ мӮ¬н•ӯмқҖ abstractлЎң лӮЁк№Җ.

вң… к·ёлҹј мқён„°нҺҳмқҙмҠӨлҘј м¶”мғҒ нҒҙлһҳмҠӨк°Җ вҖңмғҒмҶҚл°ӣлҠ”вҖқлӢӨлҠ” кұҙ?

public interface Animal {
    void sound();
    void eat();
}

public abstract class AbstractAnimal implements Animal {
    // мқјл¶Җл§Ң кө¬нҳ„н•Ё
    @Override
    public void eat() {
        System.out.println("This animal eats food.");
    }

    // sound()лҠ” кө¬нҳ„ м•Ҳ н•Ё вҶ’ мһҗмӢқ нҒҙлһҳмҠӨк°Җ кө¬нҳ„н•ҳкІҢ л§Ңл“Ұ
}

рҹ‘ү мқҙл ҮкІҢ м“°лҠ” мқҙмң ?

  1. кіөнҶө лЎңм§Ғмқ„ м¶”мғҒ нҒҙлһҳмҠӨм—җм„ң н•ң лІҲл§Ң кө¬нҳ„н•ҳкі ,
  2. к°Ғ м„ңлёҢ нҒҙлһҳмҠӨлҠ” лӮҳлЁём§Җ мқён„°нҺҳмқҙмҠӨ кё°лҠҘл§Ң кө¬нҳ„н•ҳл©ҙ лҗҳлӢҲк№Ң мҪ”л“ңк°Җ лҚ” к№”лҒ”н•ҙм§„лӢӨ.

рҹ’Ў Springм—җм„ң мһҗмЈј м“°мқҙлҠ” мқҙмң 

  • мҳҲлҘј л“Өм–ҙ, м—¬лҹ¬ м„ңл№„мҠӨ нҒҙлһҳмҠӨк°Җ кіөнҶөм Ғмқё мһ‘м—…(мҳҲ: лЎңк№…, м—җлҹ¬ мІҳлҰ¬ л“ұ)мқ„ н•ҙм•ј н•  л•Ң:
public interface MyService {
    void doSomething();
}

public abstract class AbstractMyService implements MyService {
    protected void log(String msg) {
        System.out.println("[LOG] " + msg);
    }
}

вҶ’ мқҙл ҮкІҢ м¶”мғҒ нҒҙлһҳмҠӨм—җм„ң кіөнҶө кё°лҠҘмқ„ л„Јм–ҙл‘җл©ҙ, мӢӨм ң м„ңл№„мҠӨ нҒҙлһҳмҠӨм—җм„ңлҠ” doSomething()л§Ң кө¬нҳ„н•ҳл©ҙ лҸјм„ң мң м§ҖліҙмҲҳк°Җ мү¬мӣҢм§„лӢӨ.

рҹ”Ғ м •лҰ¬н•ҳмһҗл©ҙ!

  • мқён„°нҺҳмқҙмҠӨ: вҖңмқҙ кё°лҠҘмқҖ кјӯ л§Ңл“Өм–ҙм•ј н•ҙ!вҖқлқјкі  к°•м ң.
  • м¶”мғҒ нҒҙлһҳмҠӨ: вҖңкіөнҶө мҪ”л“ңлҠ” м—¬кё° мһҲм–ҙ. л„ҲлҠ” лӮҳлЁём§Җл§Ң кө¬нҳ„н•ҙ!вҖқ
  • мқён„°нҺҳмқҙмҠӨлҘј м¶”мғҒ нҒҙлһҳмҠӨк°Җ мғҒмҶҚл°ӣлҠ” кІғ: кіөнҶө кё°лҠҘмқҖ м¶”мғҒ нҒҙлһҳмҠӨк°Җ м ңкіөн•ҳкі , лӮҳлЁём§ҖлҠ” мһҗмӢқ нҒҙлһҳмҠӨм—җкІҢ мң„мһ„н•ҳлҠ” к№”лҒ”н•ң кө¬мЎ°.

class, interface, abstract classлҘј нҶөн•ң м„Өкі„ мҳҲмӢң

// мқён„°нҺҳмқҙмҠӨ
interface Flyable {
    void fly();
}

interface Attackable {
    void attack();
}

// м¶”мғҒ нҒҙлһҳмҠӨ
abstract class Character {
    void move() {
        System.out.println("мқҙлҸҷ мӨ‘...");
    }

    abstract void specialSkill();
}

// кө¬мІҙ нҒҙлһҳмҠӨ
class Warrior extends Character implements Attackable {
    public void attack() {
        System.out.println("кІҖмңјлЎң кіөкІ©!");
    }

    public void specialSkill() {
        System.out.println("л¶„л…ё л°ңлҸҷ!");
    }
}

class Fairy extends Character implements Flyable {
    public void fly() {
        System.out.println("нӣЁнӣЁ лӮ кё°~");
    }

    public void specialSkill() {
        System.out.println("л§ҲлІ• мӮ¬мҡ©!");
    }
}
  • мқҙлҹ° мӢқмңјлЎң н•ҳл©ҙ кіөнҶө кё°лҠҘмқҖ м¶”мғҒ нҒҙлһҳмҠӨ, нҠ№мҲҳ лҠҘл ҘмқҖ мқён„°нҺҳмқҙмҠӨлЎң мЎ°н•©н•ҙм„ң мң м—°н•ң м„Өкі„к°Җ к°ҖлҠҘ.
