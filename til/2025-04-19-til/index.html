1. Omit

  • лң»: л¬ҙм–ёк°ҖлҘј м ңмҷён•ҳлӢӨ, л№јлӢӨлҠ” мқҳлҜё
  • к°ңл°ңм—җм„ңлҠ”? TypeScriptлӮҳ Java л“ұм—җм„ң к°қмІҙлҘј лӢӨлЈ° л•Ң, нҠ№м • н•„л“ңл§Ң м ңмҷён•ң мғҲ нғҖмһ…мқҙлӮҳ мғҲ к°қмІҙлҘј л§Ңл“Ө л•Ң мӮ¬мҡ©
  • мҳҲмӢң (JavaScript/TypeScript): 
    type User = { id: number; name: string; password: string };
type PublicUser = Omit<User, "password">;

    вҶ’ PublicUserлҠ” Userм—җм„ң passwordл§Ң л№ м§„ нғҖмһ…мқҙлӢӨ.

  • Javaм—җм„ңлҠ” TypeScriptмІҳлҹј Omitкё°лҠҘмқҙ м§Ғм ‘ мһҲм§ҖлҠ” м•ҠмқҢ.
    • н•ҳм§Җл§Ң вҖңDTOлҘј л”°лЎң л§Ңл“Өм–ҙм„ң н•„мҡ”н•ң н•„л“ңл§Ң лӮЁкё°лҠ” л°©мӢқвҖқмңјлЎң л№„мҠ·н•ҳкІҢ мӮ¬мҡ©
    • мҰү, Javaм—җм„ң OmitмқҙлһҖ вҖңн•„мҡ”м—ҶлҠ” н•„л“ңлҘј л№јкі  мғҲ нҒҙлһҳмҠӨлҘј л§Ңл“ңлҠ” кІғвҖқмңјлЎң н•ҙм„қн•Ё

вң… м •лҰ¬:
нҠ№м • мҶҚм„ұмқ„ л№јкі  лӢӨлЈЁлҠ” кІғмқ„ вҖңOmitвҖқмқҙлқјкі  н•ңлӢӨ.

2. Clean URL (Path Variable)

  • лң»: мЈјмҶҢ(URL)мқ„ к№”лҒ”н•ҳкІҢ, мқҳлҜё мһҲкІҢ л§Ңл“  кІғ.

  • к°ңл°ңм—җм„ңлҠ”?
    лҰ¬мҶҢмҠӨ(лҚ°мқҙн„°) мӢқлі„мқ„ URL кІҪлЎңлЎң н‘ңнҳ„н•ҳлҠ” кұё л§җн•ңлӢӨ.
    (нҢҢлқјлҜён„°лҘј кІҪлЎңм—җ к№”лҒ”н•ҳкІҢ л„ЈлҠ”лӢӨ.)

  • мҳҲмӢң: 
    GET /users/123

    вҶ’ 123лІҲ мӮ¬мҡ©мһҗлҘј мЎ°нҡҢн•ҳлҠ” мҡ”мІӯ (Path Variable = 123)

вң… м •лҰ¬:
/detail?id=123мІҳлҹј м§Җм Җл¶„н•ҳкІҢ н•ҳм§Җ м•Ҡкі , /detail/123мІҳлҹј
лҰ¬мҶҢмҠӨлҘј мЈјмҶҢ м•Ҳм—җ мһҗм—°мҠӨлҹҪкІҢ н‘ңнҳ„н•ҳлҠ” кІҢ Clean URL

3. Query String (Search Parameter)

  • лң»: ? л’Өм—җ л¶ҷлҠ” нӮӨ-к°’ мҢҚ.
  • к°ңл°ңм—җм„ңлҠ”?
    лҚ°мқҙн„° кІҖмғү, н•„н„°л§Ғ, мҳөм…ҳ м„Өм • л“ұм—җ мЈјлЎң мӮ¬мҡ©.
  • мҳҲмӢң: 
    GET /products?category=shoes&sort=price

    вҶ’ м№ҙн…Ңкі лҰ¬=мӢ л°ң, к°ҖкІ©мҲң м •л ¬мқҙлқјлҠ” мқҳлҜё.

вң… м •лҰ¬:
кІҖмғүмЎ°кұҙмқҙлӮҳ мҳөм…ҳмқ„ м „лӢ¬н•  л•Ң
?key=value&key2=value2мІҳлҹј л¶ҷмқҙлҠ” кІҢ Query String

4. HTTP Header + JWT

  • HTTP Header:

    • мҡ”мІӯ(Request)мқҙлӮҳ мқ‘лӢө(Response)мқҳ м¶”к°Җ м •ліҙлҘј лӢҙлҠ” кіі.
    • мҳҲ: мқёмҰқ м •ліҙ, мҡ”мІӯ нҸ¬л§·, нҶ нҒ° л“ұ.
  • JWT (JSON Web Token):
    • лЎңк·ёмқён•ң мӮ¬мҡ©мһҗмқём§Җ мҰқлӘ…н•ҳлҠ” нҶ нҒ°.
    • Header + Payload + SignatureлЎң кө¬м„ұлҗңлӢӨ.
    • ліҙнҶө HTTP Header м•Ҳм—җ л„Јм–ҙм„ң м„ңлІ„м—җ ліҙлғ„.
  • мҳҲмӢң: 
    Authorization: Bearer eyJhbGciOiJIUzI1...

    вҶ’ мҡ”мІӯ н—ӨлҚ”м—җ JWTлҘј лӢҙм•„ ліҙлӮёлӢӨ.

вң… м •лҰ¬:
JWTлҠ” лЎңк·ёмқё нҶ нҒ°мқҙкі ,
HTTP Headerм—җ л„Јм–ҙм„ң м„ңлІ„м—җ мқёмҰқм •ліҙлҘј м „лӢ¬н•ңлӢӨ.

5. Tuple, POJO, VO, Instance

  • Tuple:
    • м—¬лҹ¬ лҚ°мқҙн„°(нғҖмһ… лӢӨлҘј мҲҳ мһҲмқҢ)лҘј кі м •лҗң мҲңм„ңлЎң л¬¶мқҖ кІғ.
    • мЈјлЎң TypeScript, Python л“ұм—җм„ң мӮ¬мҡ©.
    • мҳҲ: (1, "apple")
  • POJO (Plain Old Java Object):
    • нҠ№лі„н•ң к·ңм№ҷ м—Ҷмқҙ, к·ёлғҘ мҲңмҲҳн•ң Java к°қмІҙ.
    • @Entity, @Component мқҙлҹ° кұ° м—ҶлҠ” мӢ¬н”Ңн•ң к°қмІҙлҘј л§җн•Ё.
  • VO (Value Object):
    • вҖңк°’вҖқ мһҗмІҙлҘј мқҳлҜён•ҳлҠ” к°қмІҙ.
    • мқҙлҰ„мқҙ к°ҷмңјл©ҙ к°ҷмқҖ кІғмңјлЎң м·Ёкёүн•ҳкі , ліҖн•ҳм§Җ м•ҠлҠ”(immutable) кІҢ мқјл°ҳм ҒмқҙлӢӨ.
    • мҳҲ: Money, Address
  • Instance:
    • нҒҙлһҳмҠӨлЎңл¶Җн„° мӢӨм ңлЎң л§Ңл“Өм–ҙм§„ л©”лӘЁлҰ¬ мғҒмқҳ к°қмІҙ.
    • мҳҲ: new User("sungwoo") вҶ’ User мқёмҠӨн„ҙмҠӨ

вң… м •лҰ¬:
| мҡ©м–ҙ | м„ӨлӘ… | |:вҖ”-|:вҖ”-| | Tuple | м„ңлЎң лӢӨлҘё нғҖмһ…мқ„ мҲңм„ңлҢҖлЎң л¬¶мқҖ лҚ°мқҙн„° | | POJO | м•„л¬ҙкІғлҸ„ нҠ№лі„н•ҳм§Җ м•ҠмқҖ мҲңмҲҳ Java к°қмІҙ | | VO | к°’(ліҖн•ҳм§Җ м•ҠлҠ” мқҳлҜё лӢЁмң„) мӨ‘мӢ¬мқҳ к°қмІҙ | | Instance | нҒҙлһҳмҠӨлЎң мӢӨм ң мғқм„ұн•ң к°қмІҙ |

6. Soft Delete

  • лң»: лҚ°мқҙн„°лҘј м§„м§ңлЎң мӮӯм ңн•ҳм§Җ м•Ҡкі , мӮӯм ңлҗң кІғмІҳлҹј н‘ңмӢңл§Ң н•ҳлҠ” кІғ.

  • к°ңл°ңм—җм„ңлҠ”? DBм—җ is_deleted к°ҷмқҖ м»¬лҹјмқ„ л§Ңл“Өм–ҙм„ң, мӮӯм ң лҢҖмӢ  н”Ңлһҳк·ёл§Ң мј лӢӨ.

  • мҳҲмӢң (MySQL н…Ңмқҙлё”): 
    UPDATE users SET is_deleted = true WHERE id = 123;

    вҶ’ мӢӨм ңлЎңлҠ” лҚ°мқҙн„°к°Җ лӮЁм•„мһҲм§Җл§Ң, мЎ°нҡҢн•  л•ҢлҠ” is_deleted = falseмқё кІғл§Ң ліҙм—¬мӨҖлӢӨ.

вң… м •лҰ¬:
Soft Delete = мӮӯм ң н‘ңмӢңл§Ң н•ҳкі  мӢӨм ң лҚ°мқҙн„°лҠ” лӮЁкІЁл‘җлҠ” кІғ
(ліөкө¬н•ҳкұ°лӮҳ кё°лЎқмқ„ лӮЁкёё л•Ң мң мҡ©)
