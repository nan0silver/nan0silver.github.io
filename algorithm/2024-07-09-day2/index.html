мң нҒҙлҰ¬л“ң нҳём ңлІ•

  • л‘җ мҲҳк°Җ м„ңлЎң мғҒлҢҖл°© мҲҳлҘј лӮҳлҲ„м–ҙ мӣҗн•ҳлҠ” мҲҳлҘј кө¬н•ҳлҠ” кІғ
  • GCD (Greatest Common Divisor) мөңлҢҖкіөм•ҪмҲҳ
    • л‘җ мһҗм—°мҲҳ a, bм—җ лҢҖн•ҙм„ң (a > b) aлҘј bлЎң лӮҳлҲҲ лӮҳлЁём§ҖлҘј rмқҙлқјкі  н•ҳл©ҙ
      • aмҷҖ bмқҳ мөңлҢҖкіөм•ҪмҲҳлҠ” bмҷҖ rмқҳ мөңлҢҖкіөм•ҪмҲҳмҷҖ лҸҷмқј
    • мқҙ м„ұм§Ҳм—җ л”°лқј, bлҘј rлЎң лӮҳлҲҲ лӮҳлЁём§Җ rвҖҷлҘј кө¬н•ҳкі ,
    • лӢӨмӢң rмқ„ rвҖҷлЎң лӮҳлҲҲ лӮҳлЁём§ҖлҘј кө¬н•ҳлҠ” кіјм •мқ„ л°ҳліөн•ҳм—¬ лӮҳлЁём§Җк°Җ 0мқҙ лҗҳм—Ҳмқ„л•Ң лӮҳлҲ„лҠ” мҲҳк°Җ aмҷҖ bмқҳ мөңлҢҖкіөм•ҪмҲҳ
    • мҳҲмӢң
      • 1071кіј 1029мқҳ мөңлҢҖкіөм•ҪмҲҳ кө¬н•ҳкё°
        • 1071 % 1029 = 42
        • 1029 % 42 = 21
        • 42 % 21 = 0
        • 21мқҙ 1071кіј 1029мқҳ мөңлҢҖкіөм•ҪмҲҳ
  • LCM (Least Common Multiple) мөңмҶҢкіөл°°мҲҳ
    • LCM(a, b) = a * b / GCD(a, b)
    • м–ҙл– н•ң л‘җ мҲҳмқҳ кіұмқҖ, к·ё л‘җ мҲҳмқҳ мөңлҢҖкіөм•ҪмҲҳмҷҖ мөңмҶҢкіөл°°мҲҳмқҳ кіұкіј к°ҷлӢӨ
  • cppмқҳ gcd, lcm н•ЁмҲҳлҠ” c++17л¶Җн„° м§Җмӣҗ
    • numeric лӘЁл“Ҳ
    • ліҙнҶө мҪ”н…Ңм—җм„ң c++17 мӮ¬мҡ©
