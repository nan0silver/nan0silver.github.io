[Algorithm] лӢӨмқөмҠӨнҠёлқј м•Ңкі лҰ¬мҰҳ

21 Feb 2026 in Algorithm Last modified at: 2026-02-25

лӢӨмқөмҠӨнҠёлқј м•Ңкі лҰ¬мҰҳм—җ лҢҖн•ҙ м •лҰ¬н•ҙліҙм•ҳмҠөлӢҲлӢӨ. н•ң м •м җм—җм„ң лӘЁл“  м •м җк№Ңм§Җмқҳ мөңлӢЁ кІҪлЎңлҘј кө¬н•ҳлҠ” лҢҖн‘ң м•Ңкі лҰ¬мҰҳмһ…лӢҲлӢӨ.

мөңлӢЁ кІҪлЎң мӢңлҰ¬мҰҲ вҖ“ н”ҢлЎңмқҙл“ң-мӣҢм…ң(лӘЁл“  мҢҚ), лІЁл§Ң-нҸ¬л“ң(мқҢмҲҳ к°„м„ В·мқҢмҲҳ мӮ¬мқҙнҒҙ)м—җ мқҙм–ҙ, н•ң м¶ңл°ңм җм—җм„ң лӮҳлЁём§Җ лӘЁл“  м •м җк№Ңм§Җмқҳ мөңлӢЁ кІҪлЎңлҘј л№ лҘҙкІҢ кө¬н•ҳлҠ” м•Ңкі лҰ¬мҰҳмқҙлӢӨ.

  1. лӢӨмқөмҠӨнҠёлқјлһҖ?
  2. н•өмӢ¬ к°ңл…җ мқҙн•ҙн•ҳкё°
  3. к°„м„  мҷ„нҷ”(Relaxation)
  4. мҡ°м„ мҲңмң„ нҒҗмҷҖ лҸҷмһ‘ кіјм •
  5. мҪ”л“ң кө¬нҳ„
  6. мӢңк°„ліөмһЎлҸ„мҷҖ кіөк°„ліөмһЎлҸ„
  7. лІЁл§Ң-нҸ¬л“ңВ·н”ҢлЎңмқҙл“ң-мӣҢм…ңкіјмқҳ л№„көҗ
  8. мӢӨм „ л¬ём ң м Ғмҡ©

рҹҺҜ 1пёҸвғЈ лӢӨмқөмҠӨнҠёлқјлһҖ?

рҹ’Ў н•ң л¬ёмһҘ м •мқҳ

вҖңмқҢмҲҳ к°„м„ мқҙ м—Ҷмқ„ л•Ң, н•ң м¶ңл°ңм җм—җм„ң лӘЁл“  м •м җк№Ңм§Җмқҳ мөңлӢЁ кІҪлЎңлҘј кө¬н•ҳлҠ” к·ёлҰ¬л”” м•Ңкі лҰ¬мҰҳвҖқ

рҹ’Ў м–ём ң мӮ¬мҡ©н•ҳлҠ”к°Җ?

  • н•ң м •м җ(м¶ңл°ңм җ)м—җм„ң лӮҳлЁём§Җ лӘЁл“  м •м җк№Ңм§Җмқҳ мөңлӢЁ кұ°лҰ¬к°Җ н•„мҡ”н•  л•Ң
  • к°„м„  к°ҖмӨ‘м№ҳк°Җ лӘЁл‘җ 0 мқҙмғҒ(л№„мқҢмҲҳ)мқј л•Ң
  • м •м җВ·к°„м„ мқҙ л§Һм•„м„ң н”ҢлЎңмқҙл“ң-мӣҢм…ң O(VВі)мқ„ м“°кё° м–ҙл Өмҡё л•Ң

рҹ’Ў нҠ№м§•

нҠ№м§• м„ӨлӘ…
мӢңк°„ліөмһЎлҸ„ O((V+E)log V) (мҡ°м„ мҲңмң„ нҒҗ мӮ¬мҡ© мӢң)
кіөк°„ліөмһЎлҸ„ O(V)
мқҢмҲҳ к°„м„  л¶Ҳк°ҖлҠҘ (мһҳлӘ»лҗң кІ°кіј к°ҖлҠҘ)
к·ёлһҳн”„ мў…лҘҳ л°©н–Ҙ/л¬ҙл°©н–Ҙ лӘЁл‘җ к°ҖлҠҘ
кө¬нҳ„ лӮңмқҙлҸ„ мӨ‘к°„ (мҡ°м„ мҲңмң„ нҒҗ нҷңмҡ©)
м„ұкІ© к·ёлҰ¬л”” (л§Ө мҲңк°„ к°ҖмһҘ к°Җк№Ңмҡҙ м •м җ нҷ•м •)

рҹ§  2пёҸвғЈ н•өмӢ¬ к°ңл…җ мқҙн•ҙн•ҳкё°

рҹ’Ў м•Ңкі лҰ¬мҰҳмқҳ ліём§Ҳ

"л§Ө лӢЁкі„л§ҲлӢӨ 'к°ҖмһҘ кұ°лҰ¬к°Җ м§§мқҖ м •м җ'мқ„ н•ҳлӮҳм”© нҷ•м •н•ҳкі ,
к·ё м •м җмқ„ нҶөн•ҙ к°Ҳ мҲҳ мһҲлҠ” мқҙмӣғл“Өмқҳ кұ°лҰ¬лҘј к°ұмӢ н•ҙ лӮҳк°ҖлҠ” м•Ңкі лҰ¬мҰҳ"

рҹ’Ў dist[v]мқҳ мқҳлҜё

dist[v] = м¶ңл°ңм җ(start)мңјлЎңл¶Җн„°
          м •м җ vк№Ңм§Җ лҸ„лӢ¬н•ҳлҠ” нҳ„мһ¬к№Ңм§Җмқҳ мөңмҶҢ л№„мҡ©
  • н•ң лІҲ нҷ•м •(л°©л¬ё мІҳлҰ¬)н•ң м •м җмқҳ dist к°’мқҖ лҚ” мқҙмғҒ л°”лҖҢм§Җ м•ҠлҠ”лӢӨ.
  • мқҢмҲҳ к°„м„ мқҙ м—Ҷкё° л•Ңл¬ём—җ, вҖңм§ҖкёҲ к°ҖмһҘ мһ‘мқҖ distлҘј к°Җм§„ м •м җвҖқмқҖ мқҙлҜё мөңлӢЁ кұ°лҰ¬к°Җ ліҙмһҘлҗңлӢӨ.

рҹ’Ў к·ёлҰ¬л””н•ң м„ нғқ

  • вҖңнҳ„мһ¬ мӢңм җм—җм„ң distк°Җ к°ҖмһҘ мһ‘мқҖ м •м җвҖқмқ„ кіЁлқјм„ң нҷ•м •н•ңлӢӨ.
  • к·ё м •м җмқ„ кұ°міҗ к°ҖлҠ” кІҪлЎңк°Җ лҚ” м§§м•„м§Ҳ мҲҳ мһҲлҠ” мқҙмӣғл§Ң мҷ„нҷ”(relax)н•ңлӢӨ.
  • мқҙл ҮкІҢ н•ҳл©ҙ к°Ғ м •м җмқ„ мөңлҢҖ н•ң лІҲм”©л§Ң нҷ•м • мІҳлҰ¬н•ҳл©ҙ лҗҳкі , мҡ°м„ мҲңмң„ нҒҗлЎң вҖңк°ҖмһҘ мһ‘мқҖ distвҖқлҘј л№ лҘҙкІҢ кәјлӮј мҲҳ мһҲлӢӨ.

рҹ”Ғ 3пёҸвғЈ к°„м„  мҷ„нҷ”(Relaxation)

рҹ’Ў к°„м„  мҷ„нҷ”лһҖ?

вҖңк°„м„  u вҶ’ vлҘј мқҙмҡ©н–Ҳмқ„ л•Ң, м¶ңл°ңм җм—җм„ң vк№Ңм§Җ к°ҖлҠ” кұ°лҰ¬к°Җ лҚ” м§§м•„м§Җл©ҙ dist[v]лҘј к°ұмӢ н•ҳлҠ” кіјм •вҖқ

(лІЁл§Ң-нҸ¬л“ң, н”ҢлЎңмқҙл“ң-мӣҢм…ңм—җм„ң м“°лҠ” кІғкіј к°ҷмқҖ к°ңл…җмқҙлӢӨ.)

рҹ’Ў мҷ„нҷ” мЎ°кұҙ

if (dist[v] > dist[u] + w) {
    dist[v] = dist[u] + w;
    // мҡ°м„ мҲңмң„ нҒҗ мӮ¬мҡ© мӢң: (dist[v], v) лӢӨмӢң л„Јкұ°лӮҳ к°ұмӢ 
}
  • u: л°©кёҲ нҷ•м •н•ң м •м җ
  • v: uмқҳ мқҙмӣғ м •м җ
  • w: к°„м„  (u, v)мқҳ к°ҖмӨ‘м№ҳ

рҹ“Ң 4пёҸвғЈ мҡ°м„ мҲңмң„ нҒҗмҷҖ лҸҷмһ‘ кіјм •

рҹ’Ў мҷң мҡ°м„ мҲңмң„ нҒҗлҘј м“°лҠ”к°Җ?

  • л§ӨлІҲ вҖңdistк°Җ к°ҖмһҘ мһ‘мқҖ м •м җвҖқмқ„ кіЁлқјм•ј н•ңлӢӨ.
  • л°°м—ҙлЎң л§ӨлІҲ мөңмҶҹк°’мқ„ м°ҫмңјл©ҙ O(V) вҶ’ м „мІҙ O(VВІ).
  • мҡ°м„ мҲңмң„ нҒҗ(мөңмҶҢ нһҷ)лҘј м“°л©ҙ мөңмҶҹк°’ м¶”м¶ң O(log V), к°„м„ лӢ№ мөңлҢҖ 1лІҲ к°ұмӢ  вҶ’ O((V+E)log V).

рҹ’Ў лҸҷмһ‘ нқҗлҰ„ (мҡ”м•Ҫ)

  1. м¶ңл°ңм җ startмқҳ dist[start] = 0, лӮҳлЁём§ҖлҠ” л¬ҙн•ңлҢҖ(INF)лЎң мҙҲкё°нҷ”.
  2. мҡ°м„ мҲңмң„ нҒҗм—җ (0, start) л„Јкё°. (л№„мҡ©, м •м җ лІҲнҳё)
  3. нҒҗк°Җ л№Ң л•Ңк№Ңм§Җ:
    • л№„мҡ©мқҙ к°ҖмһҘ мһ‘мқҖ (d, u) кәјлӮҙкё°.
    • мқҙлҜё d > dist[u] мқҙл©ҙ мҠӨнӮө (мҳҲм „м—җ л„ЈмқҖ м“ёлӘЁм—ҶлҠ” м •ліҙ).
    • uлҘј нҷ•м •н–ҲлӢӨкі  ліҙкі , uмҷҖ м—°кІ°лҗң к°Ғ к°„м„  (u вҶ’ v, к°ҖмӨ‘м№ҳ w)м—җ лҢҖн•ҙ мҷ„нҷ”.
    • мҷ„нҷ”лЎң dist[v]к°Җ мӨ„м–ҙл“Өл©ҙ (dist[v], v) лҘј нҒҗм—җ л„Јкё°.
  4. мў…лЈҢ нӣ„ dist[]к°Җ м¶ңл°ңм җм—җм„ң к°Ғ м •м җк№Ңм§Җмқҳ мөңлӢЁ кұ°лҰ¬.

рҹ’Ў вҖңнҷ•м •вҖқмқҳ мқҳлҜё

  • мқҢмҲҳ к°„м„ мқҙ м—ҶмңјлҜҖлЎң, нҳ„мһ¬ distк°Җ к°ҖмһҘ мһ‘мқҖ м •м җмқҖ лҚ” мқҙмғҒ лӢӨлҘё кІҪлЎңлЎң лҚ” м§§м•„м§Ҳ мҲҳ м—ҶлӢӨ.
  • к·ёлһҳм„ң к·ё м •м җмқҖ н•ң лІҲл§Ң кәјлӮҙм„ң мқҙмӣғл§Ң мҷ„нҷ”н•ҳл©ҙ лҗҳкі , лӢӨмӢң нҒҗм—җ л„Јмқ„ н•„мҡ” м—ҶлӢӨ.
  • d > dist[u] мқё кІҪмҡ°лҠ” вҖңмҳҲм „м—җ л„Јм—ҲлҚҳ лҚ” лӮҳмҒң к°’вҖқмқҙлҜҖлЎң л¬ҙмӢңн•ңлӢӨ.

рҹ’» 5пёҸвғЈ мҪ”л“ң кө¬нҳ„

рҹ’Ў мқём ‘ лҰ¬мҠӨнҠё + мҡ°м„ мҲңмң„ нҒҗ (Java)

import java.util.*;

class Dijkstra {
    static final int INF = 987654321;
    List<int[]>[] graph;  // graph[v] = { {next, cost}, ... }
    int[] dist;
    int V;

    public int[] dijkstra(int start) {
        dist = new int[V + 1];
        Arrays.fill(dist, INF);
        dist[start] = 0;

        // (л№„мҡ©, м •м җ) - л№„мҡ© кё°мӨҖ мөңмҶҢ нһҷ
        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
        pq.offer(new int[]{0, start});

        while (!pq.isEmpty()) {
            int[] cur = pq.poll();
            int d = cur[0], u = cur[1];

            if (d > dist[u]) continue;  // мқҙлҜё лҚ” мўӢмқҖ кІҪлЎңлЎң к°ұмӢ лҗң кІҪмҡ° мҠӨнӮө

            for (int[] edge : graph[u]) {
                int v = edge[0], w = edge[1];
                if (dist[v] > dist[u] + w) {
                    dist[v] = dist[u] + w;
                    pq.offer(new int[]{dist[v], v});
                }
            }
        }

        return dist;
    }
}

рҹ’Ў мқём ‘ н–үл ¬ лІ„м „ (м •м җ мҲҳк°Җ мһ‘мқ„ л•Ң)

  • м •м җмқҙ м Ғкі  к°„м„ мқҙ л§Һмқ„ л•ҢлҠ” мқём ‘ н–үл ¬ + л§ӨлІҲ мөңмҶҹк°’ нғҗмғүмңјлЎңлҸ„ кө¬нҳ„ к°ҖлҠҘ.
  • мӢңк°„ліөмһЎлҸ„ O(VВІ). н”ҢлЎңмқҙл“ң-мӣҢм…ңліҙлӢӨлҠ” Vк°Җ нҒҙ л•Ң мң лҰ¬н•  мҲҳ мһҲлӢӨ.

вҸұпёҸ 6пёҸвғЈ мӢңк°„ліөмһЎлҸ„мҷҖ кіөк°„ліөмһЎлҸ„

рҹ’Ў мҡ°м„ мҲңмң„ нҒҗ лІ„м „

кө¬л¶„ ліөмһЎлҸ„ м„ӨлӘ…
мӢңк°„ліөмһЎлҸ„ O((V+E) log V) к°Ғ к°„м„  мөңлҢҖ 1лІҲ мҷ„нҷ”, PQ мӮҪмһ…/мӮӯм ң log V
кіөк°„ліөмһЎлҸ„ O(V) dist[], PQм—җ м •м җ к°ңмҲҳл§ҢнҒј

рҹ’Ў мқём ‘ н–үл ¬ + м„ нҳ• нғҗмғү лІ„м „

кө¬л¶„ ліөмһЎлҸ„ м„ӨлӘ…
мӢңк°„ліөмһЎлҸ„ O(VВІ) VлІҲ л°ҳліө, л§ӨлІҲ O(V) мөңмҶҹк°’
кіөк°„ліөмһЎлҸ„ O(VВІ) мқём ‘ н–үл ¬

вҡ–пёҸ 7пёҸвғЈ лІЁл§Ң-нҸ¬л“ңВ·н”ҢлЎңмқҙл“ң-мӣҢм…ңкіјмқҳ л№„көҗ

кө¬л¶„ лӢӨмқөмҠӨнҠёлқј лІЁл§Ң-нҸ¬л“ң н”ҢлЎңмқҙл“ң-мӣҢм…ң
лӘ©м Ғ н•ң м¶ңл°ңм җ вҶ’ лӘЁл“  м •м җ н•ң м¶ңл°ңм җ вҶ’ лӘЁл“  м •м җ лӘЁл“  мҢҚ мөңлӢЁ кІҪлЎң
мқҢмҲҳ к°„м„  л¶Ҳк°ҖлҠҘ к°ҖлҠҘ к°ҖлҠҘ (мқҢмҲҳ мӮ¬мқҙнҒҙ м ңмҷё)
мқҢмҲҳ мӮ¬мқҙнҒҙ н•ҙлӢ№ м—ҶмқҢ нғҗм§Җ к°ҖлҠҘ мһҲмңјл©ҙ мӮ¬мҡ© л¶Ҳк°Җ
мӢңк°„ліөмһЎлҸ„ O((V+E)log V) лҳҗлҠ” O(VВІ) O(VE) O(VВі)
кіөк°„ліөмһЎлҸ„ O(V) O(V) O(VВІ)
м Ғмҡ© мғҒнҷ© м¶ңл°ңм җ н•ҳлӮҳ, мқҢмҲҳ к°„м„  м—ҶмқҢ мқҢмҲҳ к°„м„ /мӮ¬мқҙнҒҙ м •м җ м Ғмқ„ л•Ң, лӘЁл“  мҢҚ

рҹ’Ў м–ём ң л¬ҙм—Үмқ„ м“ёк№Ң?

рҹ§Ә 8пёҸвғЈ мӢӨм „ л¬ём ң м Ғмҡ©

рҹ’Ў лҢҖн‘ң л¬ём ң мң нҳ•

мң нҳ• м„ӨлӘ… мҳҲмӢң нӮӨмӣҢл“ң
лӢЁмқј м¶ңл°ң мөңлӢЁ кІҪлЎң н•ң лҸ„мӢңм—җм„ң лӢӨлҘё лӘЁл“  лҸ„мӢңк№Ңм§Җ вҖң1лІҲ л…ёл“ңм—җм„ңвҖқ, вҖңн•ң м җм—җм„ңвҖқ
к°ҖмӨ‘м№ҳк°Җ м–‘мҲҳ кұ°лҰ¬, мӢңк°„, л№„мҡ©мқҙ 0 мқҙмғҒ вҖңмөңмҶҢ л№„мҡ©вҖқ, вҖңмөңлӢЁ кұ°лҰ¬вҖқ
кІҪлЎң ліөмӣҗ мөңлӢЁ кІҪлЎңм—җ нҸ¬н•Ёлҗң м •м җ/к°„м„  м¶ңл Ҙ вҖңкІҪлЎң м¶ңл ҘвҖқ, вҖңкІҪлЎң м¶”м ҒвҖқ
2м°Ёмӣҗ кІ©мһҗ м№ё мқҙлҸҷ л№„мҡ©мқ„ к°„м„ мңјлЎң л‘җкі  м Ғмҡ© вҖңлӢӨмқөмҠӨнҠёлқј + BFS/кІ©мһҗвҖқ

рҹ’Ў мІҙнҒ¬лҰ¬мҠӨнҠё

лӢӨмқөмҠӨнҠёлқјлҘј м“ё л•Ң нҷ•мқён•  кІғ:

  • мқҢмҲҳ к°„м„ мқҙ м—ҶлҠ”к°Җ? (мһҲмңјл©ҙ лІЁл§Ң-нҸ¬л“ң)
  • н•ң м¶ңл°ңм җм—җм„ң лӮҳлЁём§Җ м „мІҙк°Җ н•„мҡ”н•ңк°Җ? (лӘЁл“  мҢҚмқҙл©ҙ н”ҢлЎңмқҙл“ң-мӣҢм…ң кі л Ө)
